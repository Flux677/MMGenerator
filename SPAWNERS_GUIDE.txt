=================
1. SPAWNERS
=================

Spawners adalah spawn point tetap di lokasi tertentu.

--- SETUP SPAWNER ---

1. Buat file: plugins/MythicMobs/Spawners/spawners.yml

2. Format Basic:
```yaml
SpawnerName:
  MobName: INTERNAL_MOB_NAME
  World: world
  X: 100
  Y: 64
  Z: -50
  Radius: 5
  RadiusY: 3
  UseTimer: true
  MaxMobs: 3
  MobLevel: 5
  MobsPerSpawn: 1
  Cooldown: 60
  CooldownTimer: 120
  Warmup: 0
  CheckForPlayers: true
  PlayerRange: 40
  LeashRange: 64
  HealOnLeash: true
  ResetThreatOnLeash: true
  ShowFlames: false
  Breakable: false
  Conditions:
  - day
  - playerwithin{d=50}
```

3. Contoh Spawner Boss:
```yaml
IceWizardBoss:
  MobName: ICE_WIZARD_BOSS
  World: world
  X: 100.5
  Y: 64
  Z: -50.5
  Radius: 3
  RadiusY: 2
  UseTimer: true
  MaxMobs: 1                    # Hanya 1 boss
  Cooldown: 300                 # Respawn 5 menit
  CooldownTimer: 600            # Max 10 menit
  CheckForPlayers: true
  PlayerRange: 50               # Spawn jika player dalam 50 block
  LeashRange: 100               # Boss leash 100 block
  HealOnLeash: true             # Heal saat kembali
  Conditions:
  - night                       # Hanya spawn malam
  - inregion{r=boss_arena}      # Di region tertentu (butuh WorldGuard)
```

4. Contoh Spawner Dungeon (Multiple Mobs):
```yaml
DungeonRoom1:
  MobName: DUNGEON_MINION       # Mob yang spawn
  World: world_dungeon
  X: 50
  Y: 64
  Z: 100
  Radius: 10                    # Area spawn luas
  RadiusY: 5
  UseTimer: true
  MaxMobs: 5                    # Max 5 mobs sekaligus
  MobsPerSpawn: 2               # Spawn 2 sekaligus
  Cooldown: 30                  # Respawn cepat
  CheckForPlayers: true
  PlayerRange: 30
  LeashRange: 50
```

--- MEMBUAT SPAWNER IN-GAME ---

Command:
/mm spawners create <spawner_name> <mob_name>

Contoh:
1. Berdiri di lokasi spawn
2. /mm spawners create BossSpawner ICE_WIZARD_BOSS
3. Edit spawners.yml untuk customize

--- MANAGE SPAWNERS ---

Commands:
/mm spawners list                     # List semua spawner
/mm spawners info <spawner>          # Info spawner
/mm spawners activate <spawner>      # Aktifkan
/mm spawners disable <spawner>       # Matikan
/mm spawners kill <spawner>          # Kill semua mob dari spawner
/mm spawners set <spawner> <option> <value>  # Edit option
/mm spawners remove <spawner>        # Hapus spawner

Contoh:
/mm spawners set BossSpawner cooldown 600
/mm spawners set BossSpawner maxmobs 1

--- SPAWNER OPTIONS EXPLAINED ---

MobName: Internal name mob dari Mobs.yml
World: Nama world (world, world_nether, dll)
X, Y, Z: Koordinat spawn point
Radius: Jarak random horizontal spawn (block)
RadiusY: Jarak random vertical spawn (block)

UseTimer: true/false
  - true: Spawn berdasarkan cooldown
  - false: Spawn manual atau kondisi khusus

MaxMobs: Jumlah maksimal mob hidup dari spawner ini
MobLevel: Level mob (jika menggunakan level system)
MobsPerSpawn: Jumlah mob yang spawn sekaligus

Cooldown: Waktu tunggu spawn (detik) setelah mob mati
CooldownTimer: Waktu maksimal tunggu spawn (detik)
Warmup: Delay sebelum pertama spawn (detik)

CheckForPlayers: true/false
  - true: Hanya spawn jika ada player
  
PlayerRange: Jarak player dari spawner agar spawn (block)
LeashRange: Jarak maksimal mob dari spawn point (block)
  - Jika lewat, mob kembali ke spawn
  
HealOnLeash: true/false
  - true: Mob heal full saat kembali ke spawn
  
ResetThreatOnLeash: true/false
  - true: Reset aggro saat leash
  
ShowFlames: true/false
  - true: Tampilkan flame particles di spawn point
  
Breakable: true/false
  - true: Players bisa break spawner

Conditions: Kondisi agar spawner aktif
  - day / night
  - playerwithin{d=X}
  - biome{b=BIOME}
  - inregion{r=REGION_NAME} (butuh WorldGuard)

=================
2. RANDOM SPAWNS
=================

Random Spawns = mob spawn secara random di area luas (seperti vanilla spawn).

--- SETUP RANDOM SPAWNS ---

1. Buat file: plugins/MythicMobs/RandomSpawns.yml

2. Format Basic:
```yaml
ConfigVersion: 1.0

# Global settings
RandomSpawning:
  Enabled: true

# Per mob config
Mobs:
  INTERNAL_MOB_NAME:
    Enabled: true
    Action: ADD                 # ADD atau REPLACE vanilla spawns
    Priority: 1
    Chance: 0.3                 # 30% chance spawn
    
    Conditions:
    - day
    - inbiome PLAINS,FOREST
    - world world
    
    Worlds:
    - world
    - world_custom
    
    Biomes:
    - PLAINS
    - FOREST
    - MOUNTAINS
    
    MaxMobsPerChunk: 3
    MaxMobsPerWorld: 50
    MobLevel: 1-5
```

3. Contoh Random Spawn Normal Mob:
```yaml
Mobs:
  FOREST_GOBLIN:
    Enabled: true
    Action: ADD
    Priority: 1
    Chance: 0.2                 # 20% chance replace vanilla mob
    
    Conditions:
    - day
    - altitude{a=60to100}       # Hanya spawn di Y 60-100
    
    Worlds:
    - world
    
    Biomes:
    - FOREST
    - BIRCH_FOREST
    - DARK_FOREST
    
    MaxMobsPerChunk: 2          # Max 2 per chunk
    MaxMobsPerWorld: 100
    MobLevel: 1-10
```

4. Contoh Random Spawn Rare Boss:
```yaml
Mobs:
  WANDERING_BOSS:
    Enabled: true
    Action: ADD
    Priority: 5                 # High priority
    Chance: 0.01                # 1% chance (rare!)
    
    Conditions:
    - night
    - playerwithin{d=100}       # Ada player nearby
    
    Worlds:
    - world
    
    Biomes:
    - ALL                       # Semua biome
    
    MaxMobsPerChunk: 1
    MaxMobsPerWorld: 3          # Max 3 boss di seluruh world
    MobLevel: 20
```

--- RANDOM SPAWN OPTIONS ---

Enabled: true/false
  - Aktifkan random spawn untuk mob ini

Action: ADD / REPLACE
  - ADD: Spawn TAMBAHAN ke vanilla
  - REPLACE: GANTI vanilla mob dengan custom

Priority: 1-10
  - Semakin tinggi = prioritas spawn lebih tinggi

Chance: 0.0-1.0
  - 0.1 = 10% chance
  - 0.5 = 50% chance
  - 1.0 = 100% chance

Conditions: Kondisi spawn
  - day / night
  - inbiome BIOME_NAME
  - world WORLD_NAME
  - altitude{a=Y1toY2}
  - playerwithin{d=X}
  - weather CLEAR/RAIN/THUNDER

Worlds: List world yang diizinkan
Biomes: List biome yang diizinkan
  - Gunakan "ALL" untuk semua biome

MaxMobsPerChunk: Max mob per chunk
MaxMobsPerWorld: Max mob di seluruh world
MobLevel: Level mob (1-100 atau 1-10)

--- ADVANCED: SPAWN CONDITIONS ---

Kombinasi multiple conditions:
```yaml
Conditions:
- day
- playerwithin{d=50}
- inbiome FOREST,PLAINS
- altitude{a=50to120}
- world world
- !incombat                     # ! = NOT (tidak dalam combat)
```

Time-based spawns:
```yaml
Conditions:
- time{t=12000to24000}          # Spawn antara tick 12000-24000
- weather THUNDER               # Hanya saat thunderstorm
- moonphase FULL                # Hanya full moon
```

=================
3. TIPS & BEST PRACTICES
=================

### SPAWNERS ###

1. Boss Spawners:
   - MaxMobs: 1
   - Cooldown: 300-600 (5-10 menit)
   - LeashRange: Besar (100+)
   - CheckForPlayers: true

2. Dungeon Spawners:
   - MaxMobs: 3-10
   - Cooldown: 30-60 (cepat)
   - LeashRange: Kecil (30-50)
   - MobsPerSpawn: 2-3

3. Event Spawners:
   - UseTimer: false
   - Trigger manual via command
   - Conditions khusus

### RANDOM SPAWNS ###

1. Normal Mobs:
   - Chance: 0.1-0.3 (10-30%)
   - Action: ADD
   - MaxMobsPerWorld: 50-200

2. Rare Mobs:
   - Chance: 0.01-0.05 (1-5%)
   - Action: ADD
   - MaxMobsPerWorld: 5-20

3. Boss World:
   - Chance: 0.001-0.01 (0.1-1%)
   - MaxMobsPerWorld: 1-5
   - Conditions strict (night, biome, dll)

### PERFORMANCE ###

1. Jangan spawn terlalu banyak mob:
   - MaxMobsPerChunk: < 5
   - MaxMobsPerWorld: < 200

2. Gunakan LeashRange untuk dungeon:
   - Mob kembali jika player kabur
   - Hindari mob menumpuk jauh

3. CheckForPlayers untuk boss:
   - Hemat resources jika tidak ada player

### BALANCE ###

1. Cooldown based on difficulty:
   - Normal mob: 30-60 detik
   - Mini boss: 120-300 detik
   - Boss: 300-600 detik

2. Density control:
   - MaxMobsPerChunk: 1-3
   - Spawner radius: 5-15 block

3. Leash untuk fairness:
   - Player tidak bisa kite boss keluar arena
   - Mob heal saat leash (HealOnLeash: true)

=================
4. TROUBLESHOOTING
=================

### Spawner Tidak Spawn ###

1. Check /mm spawners info <spawner>
   - Lihat status Active/Disabled
   - Check MaxMobs sudah penuh atau belum

2. Check conditions:
   - PlayerRange: Ada player dalam range?
   - Conditions: Day/night match?
   - World: Benar world?

3. Check mob config:
   - /mm mobs list | grep MOB_NAME
   - Test manual spawn: /mm mobs spawn MOB_NAME

### Random Spawn Tidak Muncul ###

1. Check RandomSpawns.yml:
   - Enabled: true?
   - Worlds correct?
   - Biomes correct?

2. Check chance:
   - Chance terlalu kecil? (< 0.01)
   - Naikkan chance untuk testing

3. Check MaxMobsPerWorld:
   - Mungkin sudah penuh
   - /mm mobs killall MOB_NAME

### Mob Spawn Terlalu Banyak ###

1. Turunkan:
   - MaxMobsPerChunk
   - MaxMobsPerWorld
   - Chance (random spawns)

2. Tambah conditions:
   - playerwithin (spawn hanya dekat player)
   - altitude (limit area spawn)

### Performance Issues ###

1. Optimize spawners:
   - Kurangi MaxMobs
   - Perbesar Cooldown
   - Tambah LeashRange

2. Optimize random spawns:
   - Turunkan Chance
   - Kurangi MaxMobsPerWorld
   - Limit Biomes/Worlds
